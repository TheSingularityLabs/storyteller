## üé¨ **Animation Prompt Format (MANDATORY)**

**Every scene MUST include an Animation Prompt** following this exact format:

```markdown
Animation Prompt for AI Video Model:
"9:16 vertical format. Animate ONLY the elements visible in the static image. Pure white background. [Text element] already visible - animate it smoothly fading in (0-1.5s) in bold black sans-serif. [Visual element 1] already visible - animate it smoothly materializing (1-2.5s). [Visual element 2] already visible - animate it smoothly appearing (2-3.5s) and subtly pulsing (3-6s). All elements animate smoothly throughout the full 6 seconds. Do NOT add any new elements. Only animate what exists in the static image. Camera is static, centered position. Clean, minimalist aesthetic. Duration: 6 seconds. Auto-switch to next scene after 6 seconds."
```

### **Key Requirements:**
- ‚úÖ **Start with**: "Animate ONLY the elements visible in the static image"
- ‚úÖ **Reference elements as "already visible"** - describe what exists, then animate it
- ‚úÖ **Smooth, gradual animations** throughout the full 6 seconds (no "quickly" or "rapidly")
- ‚úÖ **Distributed timing** - animations happen continuously across the 6 seconds
- ‚úÖ **End with**: "Auto-switch to next scene after 6 seconds"
- ‚úÖ **No new elements** - only animate what's in the static image

### **Animation Timing Guidelines:**
- Text: 0-1.5s (smooth fade in)
- Primary visuals: 1-2.5s (smooth materialization)
- Secondary elements: 2-4s (smooth appearance)
- Continuous effects: 2.5-6s (smooth pulsing/flowing)
- All animations overlap and run throughout the full 6 seconds
